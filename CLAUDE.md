# CLAUDE.md

このリポジトリの実装ルールは `AGENTS.md` に集約しています。

Claude Code など `CLAUDE.md` を読むツールで作業する場合も、最初に以下を読んでください。

1. `AGENTS.md`
2. `PROGRESS.md`
3. `CODEBASE_GUIDE.md`
4. 変更対象に対応する実装ファイル

`docs/coding-rules.md` は責務分割や依存方向を変える時だけ読む。確認・デバッグの手順は `docs/codex-workflow.md` を参照する。

古い `CLAUDE.md` には 16 ステップ時代のデータモデルや旧ファイル構成が残っていたため、重複した実装指示はここには置きません。
