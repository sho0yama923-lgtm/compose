# チュートリアル台本

## 1. 画面説明

まず、画面の説明をします。

上部では、表示するトラックの切り替え、BPM、曲のルートやスケールを確認できます。

中央のトラックエリアには、ドラムやコードなどのトラックが並びます。ここで、それぞれのトラックに何が入っているか確認できます。

下部の再生エリアでは、曲の再生、停止、小節移動、再生範囲の変更ができます。

トラックをタップすると、そのトラックの編集画面を開けます。まず、Drumsを開いてみましょう！

エディタ画面では、どこで音を鳴らすかを決めます。

1画面には1小節分が表示されます。通常のリズムでは、1小節が16分割されています。

音符のボタンでは、これから置く音の長さを選びます。

グリッドでは、どの音をどのタイミングで鳴らすかを決めます。

エディタ画面の説明は以上です。トラック全体の画面に戻りましょう！

次は、トラックの音をオン・オフする操作を試してみましょう。

## 2. ミュート操作

トラック左上のチェックで、そのトラックの音をオン・オフできます。

まず、Drumsのチェックを外してみましょう！

次に、もう一度チェックを入れて、ドラムの音を戻しましょう！

次は、曲の再生と停止を試してみましょう。

## 3. 再生操作

再生ボタンを押して、曲を聴いてみましょう！

少し聴いたら、同じボタンを押して停止しましょう。

次は、再生する範囲を変えてみましょう。

下部の再生エリアにある小さなつまみをタップして、再生範囲の操作を表示します。

青いバーが再生範囲です。右端の青いつまみが、再生を終える位置です。黄色い「ここまで」の線まで動かしてみましょう！

再生範囲が1〜2小節目になりました。再生ボタンを押して、狭めた範囲を聴いてみましょう！

少し聴いたら、同じボタンを押して停止しましょう。

最後に、右端の青いつまみを黄色い「ここまで」の線まで動かし、再生範囲を1〜4小節目へ戻しましょう。

ここからは、残りの3〜4小節を実際に作っていきます。

今はまだ2小節目までしかないので、まずドラムのトラックを4小節目まで繰り返してみましょう。

## 4. 繰り返し設定

1〜2小節目のドラムを、3〜4小節目まで繰り返してみましょう！

まず、繰り返す元になる範囲の開始位置を決めます。1小節目の左端のバーをタップしてみましょう！

次に、繰り返す元になる範囲の終了位置を決めます。2小節目に移動しましょう。

2小節目の右端のバーをタップしてみましょう！

繰り返す元の範囲が黄色で表示されました。この1〜2小節目を、後ろの小節へ繰り返します。

次は、どこまで繰り返すかを決めます。3小節目に移動しましょう。

トラック右上の繰り返しボタンを押すと、黄色の範囲が今の小節まで繰り返されます。押してみましょう！

次に、4小節目へ移動しましょう。

もう一度、繰り返しボタンを押してみましょう！

繰り返しが入った小節は緑色で表示されます。これで、1〜2小節目のドラムを3〜4小節目まで使えるようになりました！

次は、コードを置いて曲の流れを作ってみましょう。

## 5. コード設定

前半にはC → G → Am → Emのコードが入っています。ここでは後半をF → C → F → Gにして、カノン進行を完成させます。

コードは2拍ずつ置いていきます。まず、全体画面の「コード / Piano」を開きましょう！

3小節目をF → Cにします。

まず、上のコード選択でFを選びます。

Fを1〜2拍目に置きます。1拍目をタップしましょう！

続けて、2拍目をタップしましょう！

次に、上のコード選択でCを選びます。

Cを3〜4拍目に置きます。3拍目をタップしましょう！

続けて、4拍目をタップしましょう！

3小節目がF → Cになりました。次は、4小節目をF → Gにします。

まず、4小節目へ移動しましょう。

上のコード選択でFを選びます。

Fを1〜2拍目に置きます。1拍目をタップしましょう！

続けて、2拍目をタップしましょう！

最後に、上のコード選択でGを選びます。

Gを3〜4拍目に置きます。3拍目をタップしましょう！

続けて、4拍目をタップしましょう！

後半がF → C → F → Gになりました。これでコード進行がつながりました！

次は、コードを鳴らすタイミングを決めてみましょう。

キックとコードを同じタイミングで鳴らすと、リズムにまとまりが生まれます。まず、3小節目のコードをキックに合わせてみましょう！

まず、ドラムのリズムを見るために「ドラムを参照」をタップしましょう！

次に、Kick（キック）にチェックを入れましょう！

最後に「同期」を押すと、キックと同じ位置でコードが鳴るようになります！

3小節目のコードが、キックと同じタイミングで鳴るようになりました。再生して確認してみましょう。

再生ボタンを押して、キックとコードが重なるところを聴いてみましょう！

確認できたら停止しましょう。

3小節目では、キックに合わせてコードを鳴らせました。次は4小節目で、自分で鳴らす場所を選んでみましょう。

まず、4小節目へ移動しましょう。

4小節目でコードを鳴らしたい場所を一つタップしてみましょう！

コード進行と、コードを鳴らすタイミングを設定できました！

次は、コードに合わせてメロディを作ってみましょう。

## 6. メロディ作成

最後に、コードに合わせてメロディを作ります。

音を鳴らしたい場所をタップすると、音を置けます。

グリッドには、コードに含まれる音が色付きで表示されています。迷ったときは、まず色が付いたコードトーンから選ぶと、曲になじみやすくなります。

まず、色が付いたコードトーンから一つ選んで置いてみましょう！

音を置けました。次は、置いた音を移動してみましょう。

音の高さやタイミングを変えたいときは、長押しして移動できます。置いた音を長押しして、上下または左右へ動かしてみましょう！

音を移動できました。最後に、いらない音を削除してみましょう。

音を置き直したいときは削除できます。移動した音をタップして削除してみましょう！

音の配置、移動、削除ができました。これで基本的な作曲の流れは完了です！
